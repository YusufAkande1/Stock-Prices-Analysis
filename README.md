# 📊 Stock Market Prices Analysis

This project focuses on analyzing stock market data to uncover trends, patterns, and insights using Python and Power BI.

## 🚀 Project Overview
The analysis involves data cleaning, exploratory data analysis (EDA), time series analysis, and dashboard creation to visualize stock performance and trading behavior.

## 📁 Dataset Features
- Symbol – Stock ticker
- Date – Trading date
- Open – Opening price
- High – Highest price
- Low – Lowest price
- Close – Closing price
- Volume – Trading volume

## 🛠️ Tools & Technologies
- Python (pandas, matplotlib)
- Power BI
- CSV Dataset

## 🔍 Key Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Time series analysis using 30-day moving average
- Data visualization using Python
- Interactive dashboard creation in Power BI

## 📊 Dashboard Features
- KPI cards (Average prices, total volume, etc.)
- Stock price trend analysis
- 30-day moving average visualization
- Volume comparison across companies
- Interactive filtering using slicers

## 💡 Key Insights
- Stock prices show an overall upward trend with periods of volatility
- Moving averages help identify long-term trends
- Trading volume varies significantly across stocks
- Market behavior includes bullish and bearish phases

## 📌 Conclusion
This project demonstrates how data analytics techniques can be applied to financial data to generate meaningful insights and support decision-making.
